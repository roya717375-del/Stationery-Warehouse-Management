# Stationery Warehouse Management System

## Overview
This project is a Java desktop application for managing stationery warehouses. It provides inventory management, warehouse management, item tracking, permission handling, and financial transaction recording using an SQLite database.

## Features

- Warehouse Management
- Item & Category Management
- Inventory Tracking
- Permission In / Permission Out
- Financial Transactions
- Cash Balance
- SQLite Database

## Technologies

- Java
- Swing
- JDBC
- SQLite
